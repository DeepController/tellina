# 1

find Directory -type f -mtime Timespan -exec rm {} \;

Explanation

Remove all regular files in Directory modified Timespan days ago

# 2

find Directory -user Username -mtime Timespan -exec rm {} \;

Explanation

Remove all files/directories in Directory which belong to user Username and was modified Timespan days ago.

# 3

find Directory -type f -mtime Timespan -exec rm -f {} \;

Explanation

Force remove all regular files in Directory modified Timespan days ago.

# 4

find Directory -type f -mtime Timespan -print0 | xargs -0 -I {} rm {}

Explanation

Remove all regular files in Directory modified Timespan days ago.
