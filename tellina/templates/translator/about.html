Hello,

Tellina is a natural language to bash commands translator aiming to reduce end-users' programming and learning efforts. For example, it is able to translate a sentence like "Find text file in the current folder" into a bash command find . -name "*.txt". .

The system is written in Python and uses bashlex, Tensorflow and Django. Source code is available on github.
