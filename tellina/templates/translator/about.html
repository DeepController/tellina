About

Tellina is a natural language to bash commands translator aiming to reduce end-users' programming and learning efforts. For example, it is able to translate a sentence like "Find text file in the current folder" into a bash command find . -name "*.txt". The system is written in Python and uses bashlex, Tensorflow and Django. Source code is available on github.

How to use Tellina?

Translating a command

You can perform the translation from an English sentence to a bash command by typing the sentence in the search page and click the "Translate" button. You can also search your sentence on Google using the shortcut "Search with Google" button.

Interpreting the results

When you are brought to the result page, you will see a list of result entries in the following format.

Though our goal is to build a translation system with perfect precision, we are not quite there yet :(. We designed the following interactions to help you understand the commands generated by the system. Thus, before copy-pasting the commands to run on your computer, you can try to understand the suggested commands using the instruction below so that you can confirm the correctness of a command (or modify it when it seems to be incorrect).

  • Code format: Tokens highlighted in orange are command heads, tokens highlighted in purple are flags of a command, and tokens with underline are arguments of some command or some flag. You can hover over these tokens to get their explanation based on their man page information.
  • Copy a command: You can copy a command to your clipboard by clicking the button on the right hand side of a command.
  • Explain a command: You can find the explanation of a command on the explainshell.com by clicking the button on the right hand side of a command.
